# 畢業專題-機車神盾(PHP)

## 系統特色與目的
- 結合Google地圖資訊完成自動打方向燈機制來降低車禍發生機率及違規造成的荷包損失
- 當機車被移動時，系統自動啟動攝影機錄影及發出通知
- 透過連結google雲端硬碟，可立即觀看即時資訊影像，有效找尋肇事者
- 透過GPS系統可查看車輛即時位置
